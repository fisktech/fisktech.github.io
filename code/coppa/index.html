The Children's Online Privacy Protection Act, effective April 21, 2000, applies to the online collection of personal information from children under 13. I do not collect any information from students participating in this program. However, some of the websites we will be using have the option to create an account. For that reason, I have posted links to each sites' privacy policy. Please click on the buttons to learn more about what kind of information each site collects. Note: It is completely optional to register and create an account on any of these sites. Nothing we do in this class will require it. The only real benefit is that it unlocks some content and allows the students to save their place and resume the coding exercises at home.
