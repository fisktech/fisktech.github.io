As Director of Information Technology, you have the following responsibilities:

  • Maintaining the IRHA Web Server and IRHA Website(s)
  • Assisting with IT projects to benefit Residence Hall Students
  • Forming good relationships with campus and DoR IT staff

Meeting Minutes

As Director of Information Technology, you are in charge of keeping accurate minutes at each meeting and ensuring that they are publicly accessible. We use Google Docs to facilitate this process. It allows you to share the document with your fellow exec members so that they can add things to the minutes or help you keep up. It also allows you to provide public access to the minutes in real-time.

Google Drive

All of the exec documents (agendas, minutes, etc.) are stored in a Google Drive folder. The IRHA President or Vice President will give you access to this folder. I will assume that you know your way around Google Drive, since you use it for classes.

How to take minutes

  • Unless you’re The Flash, avoid taking minutes verbatim
  • Summarize what each person says by listening for a main point and at least one piece of supporting evidence. Pay close attention to key pieces of information such as dates and numbers so that you type these accurately.
  • Start each line by noting who is talking. Ex. Friley Andrew: (Insert Main Point)
  • If you miss something, please let the others know by denoting it somehow, e.g.: Friley Andrew: My question involves [], and I think [] works well with [].
  • If you have been constantly typing, take a short break to do finger and wrist stretches and motions that extend well beyond the typical range of motion when typing. This is important. You don’t want carpal tunnel syndrome.
  • Helpful Tip: There is a keyboard shortcut to fix indentation quickly in Google Docs. CTRL + [ moves the indentation left, while CTRL + ] moves it right.

A/V Setup

As Director of IT, you are in charge of displaying relevant legislation and documents on a multimedia projector during parliament meetings. You may also be asked to show a powerpoint presentation. While the process varies slightly depending on what room Parliament meets in, here are some details that should be helpful.

How to set up projector

  1. Turn the Projector On
    • If the projector is on a cart, just press the power button on the unit. If you are in a room with a ceiling-mounted projector, there will be a control panel up front that will allow you to turn on the projector.
  2. Plug in your Laptop
    • Most projectors at ISU support HDMI and VGA input. What you use will depend on your laptop and possibly your personal preference.
  3. Check the Projector Input
    • If your screen doesn't show up on the projector when you plug in the cable, the projector is likely on the wrong input. Use the control panel or buttons on the projector to select the proper input (HDMI or VGA).

Note: If you meet in UDCC 136 you will need a key to access the control panel for the projector. The IRHA advisor or VP should give this to you before each meeting.

Remote Desktop Connection

You can access the server remotely using an RDP Client. Download for Mac or PC

RDP is a protocol created by Microsoft that allows you to access the graphical user interface (GUI) of a remote computer over a network connection. This is handy for making changes without needing physical access to the server in Friley Hall.

IIS Server Manager

IIS Server Manager is the management software for our web server. To use it, follow the steps above to RDP into the server. Once Windows loads, click the 'start' button and then search for IIS. Technical support for IIS is beyond the scope of this tutorial.

I would strongly encourage you to keep periodic backups of the website. Keeping backups will allow you to easily restore the site to a recent state if the server crashes or files become corrupt. Backup the website files and the database. The backup and restore process will only work if you have both of these components.
