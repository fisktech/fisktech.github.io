Cyclone Family Weekend

Iowa State University

http://familyweekend.iastate.edu

Site Description: This is the official website for Cyclone Family Weekend, which is an annual event held at Iowa State University in Ames, Iowa. This website was designed with naviagtion as the main focus. This is the first time I have incorporated a fullscreen menu into a website for mobile users and it works great! Since the site is focused on one event, it made sense for it to be a one-page website. The content and flow of the wesbite was designed with users in mind. Enter Site

Student Government

Iowa State University

http://stugov.iastate.edu

Site Description: This is the official website for Student Government at Iowa State University. This website was designed from the ground up to be very accesible for students and make the organization more transparent. This site was designed with mobile in mind, and content scales seemlessly for viewing on any device. Site traffic more than doubled in the first month that this site was live. This site is not complete, and new content will be added in the coming months. Enter Site

Library Website

Eagan High School

http://www.eaganhs.portal.rschooltoday.com/page/9948

Site Description: This was a re-design of the original Eagan High School Library website. The previouse website was designed almost 20 years ago and was no longer serving the needs of students and faculty. I graduated from EHS in 2014 and wanted to give back, so I volunteered to build a new wesbite. The new site is built into the main EHS website for a unified look and feel. The site is deigned to work well on any device and has already gotten lots of use.

Technology Education Course Guide

Site Description: This website was actually my final project for a class I took back in high school.
