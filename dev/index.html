Cyclone Family Weekend

Iowa State University

http://familyweekend.iastate.edu

Site Description: This is the official website for Cyclone Family Weekend, which is an annual event held at Iowa State University in Ames, Iowa. This website is a work-in-progress and will eventually have a lot more information. Once the re-design is complete, I will be converting this site to a Wordpress theme for easy content management. The site is built on the Twitter Bootstrap Framework with some custom styles & scripts. The color palette and fonts used follow the official university marketing guidlines outlined at brandmarketing.iastate.edu.

Eagan High School Library

Eagan High School

http://gofisk.com/dev/client/ehs/

Site Description: This is a re-design of the existing Eagan High School Library website. I graduated from EHS a few years ago and am working on this project just for fun (I am not getting paid). This project has a few unique challenges, such as incorporating a search function using only client-side code. My main goal is to make the site a lot more usable than it currently is. This will include a design overhaul, but also work behind the scenes to make sure that the content is relevant and accessible for students. I hope to complete this by Fall 2016.
