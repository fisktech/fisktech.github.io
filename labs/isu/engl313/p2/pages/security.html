How does this site work?

This website works by taking your input and running it through a script. This script effectively runs Luhn's Algorithm in a very efficient manner. The script will only accept valid credit card numbers with 15 or 16 digits.

This site doesn't store any data, or post it anywhere, but it easily could. It is never a good idea to enter your credit card number on an unencrypted website, such as this.

A good way to test this algorithm is by generating valid numbers using the algorithm in reverse. If you want to generate numbers for testing, you can do it right here: http://bradconte.com/cc_generator
