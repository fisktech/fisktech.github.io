Shrek The Musical

Eagan High School

http://gofisk.com/live/shrek

June 2014

Site Description: First, let me say that I had a ton of fun desiging this website. The purpose of this website was to promote Eagan High School Community Theaters' production of Shrek the Musical. There is not a lot of content on this page, but that is by design. What makes this site unique is its use of a cool JavaScript library called parallax.js. This library allows me to put elements (such as the grass and dragon) at different levels and have them react to the users cursor movements. When viewed on a mobile device with a gyroscope, the view will also move around based on the tilt of the device. This makes the website very fun to use.

Tech Ed Course Guide

Eagan High School

http://gofisk.com/live/ehstech

May 2013

Site Description: This website is an online course guide for the technology education classes offered at Eagan High School. I built this website as my final project when I took the web development course at EHS. This page is currently live on the Eagan High School website (http://www.eagan.k12.mn.us/hauenstein/CourseGuide) and is used frequently by students and parents for planning their class schedule. It provides detailed information about each class and track in the tech ed department. It also has contact information for all of the instructors.
